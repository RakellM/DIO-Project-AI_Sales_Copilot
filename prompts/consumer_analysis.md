# Consumer Analysis Prompt for Sales Reps

## Purpose
Use this prompt to analyze a customer's needs before responding.

## Prompt Template

Copy and paste this into the AI copilot:

```text
Analyze this customer interaction and tell me:

Customer information:
- Location: [city, province]
- Order history: [first time / returning / subscription]
- Stated needs: [quote the customer]
- Implied objections: [what they're NOT saying]

Based on our knowledge files, tell me:
- Which persona matches best (from personas.md)
- What is their primary objection (from objections.md)
- What track should I recommend (from product.md)
- What discount or offer makes sense
- What follow-up question should I ask

Customer said: "[paste customer quote here]"
```


## Example Usage

**Customer quote:** "I'm in Barrie. It's February. How do I know my food won't freeze?"

**AI Copilot response to this prompt:**

```test
1. Persona: Persona 5 (Skeptical First-Timer - Winter)
2. Primary objection: Trust + weather (from winter weather objections)
3. Recommended track: Start with Protein Forward (Iron Craft bowl - highest satiety, holds heat well)
4. Offer: 25% off first order + weather guarantee (full refund if cold) + no subscription
5. Follow-up question: "What time of day works best for delivery - lunch (12-1pm) or early evening (5-6pm)? I want to make sure you're home to grab the bag right away."
```


## When to Use This Prompt

- First-time customer inquiries
- Complex objections (multiple issues)
- Customer seems unsure or skeptical
- You need to identify the persona before responding
